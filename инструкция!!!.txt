1.прочитай следующую информацию:
	Типы:
	    1.Задач:
	        на кратность("crat")
	        на последнюю цифру("last_num")
	    2.Примечаний:
	        заданное кол-во чисел("NumOfNums")
	        пока не 0("while!0")
	    3.Результатов:
	        кол-во чисел, подходящих к условию("quantity")
	        сумма чисел, подходящих к условию("sum")

2.запускай PPS_console_app(кликни 2 раза ЛКМ)
3.вписывай тип задачи
4.вписывай примечание
5.вписывай результат
6.действуешь по ситуации
7.выводится код и результат задачи:
	результат должен совпадать с примером в задаче
	код можешь скопировать и проверить его в файле test.py, в любой IDE

для того чтобы всё работало должен быть установлен python версии 3.11
